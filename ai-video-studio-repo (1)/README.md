## Getting Started

### 1. Clone Repo
```bash
git clone https://github.com/YOUR-USERNAME/ai-video-studio-pro.git
cd ai-video-studio-pro
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Locally
```bash
npm run dev
```

### 4. Deploy to Vercel
- Go to [https://vercel.com/new](https://vercel.com/new)
- Import this repo
- Framework: **Next.js**
